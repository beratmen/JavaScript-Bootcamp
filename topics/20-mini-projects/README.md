# 20. Mini Projeler & Uygulamalar

## Konu Özeti
Bu konuda, önceki konuların hepsini birleştirerek küçük ama gerçekçi JavaScript projeleri yapacağız.

## Mini Projeler

### 1. Yapılacaklar Listesi (Todo App)
- Görev ekleme/silme
- Local storage ile kaydetme
- Tamamlanan görevleri işaretleme

**Teknolojiler:** DOM, Events, Local Storage

### 2. Hava Durumu Uygulaması
- API ile hava durumu verisi alma
- Şehir araması
- Sıcaklık gösterimi

**Teknolojiler:** Fetch, API, DOM

### 3. Not Defteri
- Not ekleme/silme/düzenleme
- Kategoriler
- Arama özelliği

**Teknolojiler:** Arrays, Objects, DOM, Events

### 4. Hesap Makinesi
- Temel işlemler
- Kayan noktalı hesap
- Hesap geçmişi

**Teknolojiler:** Operatörler, Functions, DOM

### 5. Quiz Uygulaması
- Soru-cevap sistemi
- Puan hesaplama
- Sonuç gösterimi

**Teknolojiler:** Arrays, Loops, Conditionals, DOM
